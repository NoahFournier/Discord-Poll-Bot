# Discord Poll Bot

## Getting Started

```
git clone https://github.com/NoahFournier/Discord-Poll-Bot.git
cd Discord-Poll-Bot
npm install
```

Once installation is finished, copy or rename `config.json.example` to `config.json`, put the file in the /src directory, and fill in your bot token from the Discord Developer Portal.

```
Commands

npm run watch --- Starts a Typescript compiler (tsc) with --watch parameter
npm run dev   --- Runs nodemon instance on compiled index.js
npm start
```
